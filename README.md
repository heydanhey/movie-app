# Movie-App

A simple movie theater web application.

## Installation

`git clone`

`cd movie-app`

`bundle`

`rake db:create`

`rake db:migrate`

`rake db:create`

`rake db:seed`

`rails s`

##### root path: `/` or `/movies`:
* shows a list of all movies playing today.
* click an available showtime to purchase tickets.
##### orders path: `/orders`:
* show a list of all orders
